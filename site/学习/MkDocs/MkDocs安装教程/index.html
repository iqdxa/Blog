    MkDocs安装教程

    安装 MkDocs

    • 使用以下命令进行安装MkDocs
    pip install mkdocs
    • 安装material主题
    pip install mkdocs-material
    • 安装完成后验证是否安装成功
    mkdocs --version

    创建 MkDocs 项目

    mkdocs new <项目名称>

    启动本地服务器

    mkdocs serve
    • 在浏览器打开 http://127.0.0.1:8000进行预览。

    生成静态文件

    mkdocs build

    使用GitHub Actions

    • 在根目录下创建文件.github/workflows/ci.yml，并加入下列代码
    name: ci
on:
  push:
    branches:
      - master
      - main
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: actions/setup-python@v2
        with:
          python-version: 3.x
      - run: pip install mkdocs-material
      - run: mkdocs gh-deploy --force

    部署到Github

    mkdocs gh-deploy
